# CSS Media Query

- untuk mengatur styling di beberapa perangkat berbeda, seperti mobile, tablet, desktop, dan lainnya.
- Untuk perangkat mobile, gunakan "max-width: 600px" sebagai referensi, dan styling di dalamnya akan aktif ketika lebar layar kurang dari 600 pixel.
- Untuk tablet (sekitar 768 pixel), gunakan "min-width: 768px" sebagai referensi, dan atur styling khusus, seperti "text-align: left" pada elemen h1.
- Penting untuk memahami batasan-batasan lebar layar untuk berbagai perangkat, seperti landscape tablets (768px), large devices (992px), dan extra-large devices (1200px atau lebih).
- Media query memungkinkan Anda membuat tampilan responsif pada website dan mengatur styling sesuai dengan kebutuhan perangkat.
- Pastikan untuk menggunakan angka yang sesuai dengan keinginan Anda dalam media query, dan pastikan menguji tampilan pada berbagai perangkat.
